🇺🇸 English

JHappyLoveAny is a Language Server Protocol (LSP) based plugin for the Eclipse IDE that provides powerful auto-completion and navigation for keys defined in .properties and XML files directly within Java source code. It eliminates the hassle of searching for key definitions manually, significantly boosting coding productivity.
🔗 For details, features, and usage, please visit GitHub.

To install the plugin, use the following update site URLs in your Eclipse IDE:

Stable Version (Main)
https://tatoo2018.github.io/JHappyLoveAny/updatesite/main/
Development Version (Develop) Experimental
https://tatoo2018.github.io/JHappyLoveAny/updatesite/develop/

Installation Guide

  1. Go to Help > Install New Software...
  2. Click Add... and paste one of the URLs above.
  3. Select JHappyLoveAny and follow the prompts.

🇨🇳 简体中文

JHappyLoveAny 是一款基于 Language Server Protocol (LSP) 的 Eclipse IDE 插件，旨在为 Java 源代码中引用的 .properties 和 XML 文件键值提供强大的自动补全与导航功能。 它免去了开发者手动查找键值定义的麻烦，从而显著提升编码生产力。

要在 Eclipse IDE 中安装，请使用以下更新站点 URL：

稳定版 (Main)
https://tatoo2018.github.io/JHappyLoveAny/updatesite/main/
开发版 (Develop) 实验性
https://tatoo2018.github.io/JHappyLoveAny/updatesite/develop/

安装步骤

  1. 在 Eclipse 中，前往 Help > Install New Software...
  2. 点击 Add... 并粘贴上述任一 URL。
  3. 勾选 JHappyLoveAny 並按提示完成安装。
