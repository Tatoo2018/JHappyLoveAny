Welcome to JHappyLoveAny

JHappyLoveAny is an Eclipse plugin designed to boost your productivity when working with Properties and XML files using Language Server Protocol (LSP).

Key Features

🚀 Smart Code Completion

Intelligently suggests keys defined in your project's XML and Properties files, reducing typos and manual searching.

🛠 Flexible Configuration

Easily define which files and which keys (via XPath) should be included in the completion list via a simple XML configuration.

Getting Started

  1. Open your Project Properties.
  2. Go to the JHappy Settings page.
  3. Create the default jhappyqueries.xml file.
  4. Start typing in your editor to see suggestions!
