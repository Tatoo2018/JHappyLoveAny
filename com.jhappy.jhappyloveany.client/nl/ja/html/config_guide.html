JHappyLoveAny 設定ガイド

このガイドでは、プロジェクト内でインテリジェントなコード補完を有効にするための jhappyqueries.xml の設定方法を説明します。

ファイルの配置場所: jhappyqueries.xml は、Eclipseプロジェクトのルートディレクトリ（直下）に配置してください。

1. XMLの基本構造

設定ファイルは、バリデーションのためのDTD定義を含むシンプルなXML形式を使用します。

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE queries [
   <!ELEMENT queries (#PCDATA|query)*>
   <!ATTLIST queries completionMatch (prefix | contains) "contains">
   <!ELEMENT query (#PCDATA)>
   <!ATTLIST query filepath CDATA #IMPLIED trim (yes | no) "yes" type (xml | properties) "xml" description CDATA #IMPLIED>
]>
<queries completionMatch="contains">
    <!-- ここにクエリを記述します -->
</queries>
