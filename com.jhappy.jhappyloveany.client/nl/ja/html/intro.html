JHappyLoveAny へようこそ

JHappyLoveAny は、LSP（Language Server Protocol）を活用して、PropertiesファイルやXMLファイルの編集効率を劇的に向上させるEclipseプラグインです。

主な機能

🚀 インテリジェントなコード補完

プロジェクト内のXMLやPropertiesファイルで定義されたキーを自動的に提案します。タイポを防ぎ、定義場所を探す手間を省きます。

🛠 柔軟なカスタマイズ

どのファイルの、どの要素（XPath指定）を補完候補にするかを、プロジェクトごとのXML設定で簡単に定義できます。

クイックスタート

  1. プロジェクトのプロパティを開きます。
  2. JHappy Settings ページを選択します。
  3. デフォルトの jhappyqueries.xml ファイルを作成します。
  4. エディタで入力を開始すると、候補が表示されます！
