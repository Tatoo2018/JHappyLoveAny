JHappyLoveAny 配置指南

本指南将介绍如何设置 jhappyqueries.xml，以便在项目中启用智能代码补全功能。

文件放置位置： 请将 jhappyqueries.xml 放置在 Eclipse 项目的根目录下。

1. XML 结构

配置文件使用简单的 XML 格式，并包含用于验证的 DTD 定义。

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE queries [
   <!ELEMENT queries (#PCDATA|query)*>
   <!ATTLIST queries completionMatch (prefix | contains) "contains">
   <!ELEMENT query (#PCDATA)>
   <!ATTLIST query filepath CDATA #IMPLIED trim (yes | no) "yes" type (xml | properties) "xml" description CDATA #IMPLIED>
]>
<queries completionMatch="contains">
    <!-- 在此处编写查询语句 -->
</queries>

2. 查询属性详解

属性名 说明
filepath 用于指定目标文件的正则表达式（例如：.*\.xml$）。
type 文件格式：xml 或 properties。
trim 是否删除提取出的键（Key）前后的空格（默认值：yes）。
description 显示在项目属性界面表格中的简短说明文字。

3. 配置示例

XML 键补全（使用 XPath）

要从 XML 属性（例如：<item id="my_key">）中提取键，请在 CDATA 部分编写 XPath 表达式。

<query filepath=".*\.xml$" type="xml"><![CDATA[//@id]]></query>

Properties 文件补全

对于标准的 .properties 文件，无需 XPath。仅需指定文件模式即可工作。

<query filepath="src/.*\.properties$" type="properties" />
