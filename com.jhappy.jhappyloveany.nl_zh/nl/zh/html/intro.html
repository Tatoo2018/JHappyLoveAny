欢迎使用 JHappyLoveAny

JHappyLoveAny 是一款利用 LSP（Language Server Protocol）技术，大幅提升 Properties 文件和 XML 文件编辑效率的 Eclipse 插件。

主要功能

🚀 智能代码补全

自动建议项目中 XML 或 Properties 文件内定义的键（Key）。有效防止输入错误，并省去查找定义位置的麻烦。

🛠 灵活的自定义配置

可以通过项目特有的 XML 配置文件，轻松定义哪些文件的哪些元素（通过 XPath 指定）应作为补全候选对象。

快速入门

  1. 打开项目的属性（Properties）界面。
  2. 选择 JHappy Settings 页面。
  3. 创建默认的 jhappyqueries.xml 文件。
  4. 在编辑器中开始输入，即可看到补全建议！
